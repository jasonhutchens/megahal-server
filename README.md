MegaHAL Server
==============

MegaHAL Server is a Rails server that allows users to chat with [MegaHAL](https://github.com/jasonhutchens/megahal), a learning chatterbot.

You can [chat with MegaHAL online](http://megahal.kranzky.com) right now!
